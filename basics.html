CSC104 HTML Basics

HTML stands for HyperText Markup Language. HTML provides the structure of the webpage by marking up the text. You can use a variety of HTML tags to build the webpage content. HTML does not have many rules, but it does have some best practices, such as using 1 and only 1 h1 tag.

History of HTML

  • HTML 1.0 - developed in early 1900's
  • HTML 2.0 - developed in 1995
  • HTML 3.2 - developed in 1997
  • HTML 4.01 - developed in 1999
  • HTML 5 - developed in 2014 (current version)

Cascading Style Sheets (CSS)

CSS is a language that is used to make HTML look better. CSS can control layout, colors, font - basically any visual element on the page can be changed from default with CSS.

History of CSS

  • CSS1 released in 1996
  • CSS2 released in 1998
  • CSS3 released in 2011 - this is the current version of CSS

Basic Structure

            selector {
                property: value;
            }
  • CSS selectors can be an HTML element, id of an HTML element, custom class
  • CSS helps with the separation of content and looks
  • CSS makes our sites easier to maintain
  • Accessibility issues can be more easily managed

3 ways to include CSS on our webpages

  • External - a seperate CSS file - this is the best practice
  • Embedded - the CSS is on 1 page - use with caution
  • Inline - the CSS is on an individual HTML element - avoid if possible
